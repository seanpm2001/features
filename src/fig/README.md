
# Fig

Installs all Fig headless components

## Example Usage

```json
"features": {
    "ghcr.io/withfig/features/fig:1": {}
}
```

## Options

| Options Id | Description              | Type   | Default Value |
| ---------- | ------------------------ | ------ | ------------- |
| version    | Select a version of fig  | string | latest        |
| channel    | Select a release channel | string | stable        |
